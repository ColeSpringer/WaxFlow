WaxFlow demo (dev mode)

Mints a signed URL via POST /sign and plays it. Direct play (format auto on an untransformed source) serves original bytes with full range support; anything else streams a live transcode where the t= box seeks sample-exact.

HLS

Mints a signed /hls/master.m3u8 URL and plays it through hls.js (fetched from a CDN; this page is dev-mode only). Safari can also take the master URL natively.

A span serves one offset range of the file as a stream in its own right: the virtual track. Samples, not seconds, because a span says which samples are this track. Leave both empty for the whole file.

Timeline (gapless queue)

Mints a play queue into a tl digest via POST /hls/timeline, then signs a master against it: several files delivered as one continuous stream, so seeking across a track boundary is just seeking. Gain must be off or a dB number here, because one stream has one gain and a queue has many tags.
